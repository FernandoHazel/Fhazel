Dino World

I worked as Unity developer in One Simple Game Studio and worked in a 5 persons team to create this demo for a publisher.


-Coded with C# to add new functionalities to the game like inventory system, UI/UX combat system and crafting.

-Applied Dotween to create interesting UI animations.

-Implemented the UI assets provided by the art team.

-Used cinemachine to record cinematics for a trailer.

-Used striptable objects to handle data like inventory, character level and items properties.

-Had daily scrum meetings with art and game design team to ensure good team communication.

-Worked closely with the client to ensure the accomplishment of his requirements.
