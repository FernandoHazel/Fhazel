AR Mobile App

This is an augmented reality application created for a mexican company with the intention of training its staff to use the machines, the application has a classroom and floor mode, which means that you can see the model of the machine on a table in classroom mode or floor mode, which means that if you are in front of the machine, it will recognize the place and guide the worker to the point of interest. This project was developed with Unity by a team of 3 people.
